# INSTALL — установка по шагам

> Можно делать самой по этим шагам, а можно отдать файл AI-помощнику (Claude и
> т.п.) и сказать: «проведи меня по этой установке шаг за шагом, делай за меня всё,
> что можешь, спрашивай только то, что нужно от меня». Hermes — сам агент, после
> установки он тоже умеет помогать.

Установка ставит **Hermes** (движок) и вкладывает в него **характер ассистента**
(этот набор). В конце ты пишешь ассистентке в Telegram.

---

## Шаг 0. Что понадобится

- Компьютер (Mac / Windows / Linux). Для работы 24/7 (Telegram отвечает всегда) —
  лучше дешёвый сервер (VPS), но для старта хватит своего компа.
- Аккаунт Telegram.
- Доступ к модели — **можно бесплатно** на старте (Nous Portal free). Claude
  подключишь потом.

## Шаг 1. Поставить Hermes (движок)

Открой [hermes-agent.nousresearch.com](https://hermes-agent.nousresearch.com) и
поставь:
- **Mac / Windows** — скачай приложение (.dmg / .exe) и установи как обычную программу.
- **Linux / сервер** — одной командой:
  ```bash
  curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash
  ```

## Шаг 2. Первый запуск и вход к модели

Запусти Hermes (приложение или команда `hermes`). Мастер настройки попросит войти
к провайдеру модели — выбери **Nous Portal** и залогинься. На бесплатном тарифе
доступна бесплатная модель — для старта её достаточно. (Поменять на Claude — Шаг 5.)

## Шаг 3. Вложить характер ассистента

Скачай этот набор (на странице репозитория **Code → Download ZIP**, распакуй).
Скопируй содержимое папки `soul/` в домашнюю папку Hermes `~/.hermes/`:

```bash
cp soul/SOUL.md ~/.hermes/SOUL.md
cp -R soul/skills/assistant ~/.hermes/skills/assistant
```

(На Windows — просто скопируй файлы в папку `.hermes` в твоём профиле
пользователя.) `SOUL.md` Hermes подхватывает сразу, без перезапуска.

## Шаг 4. Подключить Telegram

1. В Telegram напиши **@BotFather** → `/newbot` → придумай имя → скопируй **токен**.
2. Напиши **@userinfobot** → скопируй свой числовой **user id** (чтобы боту могла
   писать только ты).
3. Подключи бота в Hermes:
   ```bash
   hermes gateway setup     # выбери Telegram, вставь токен и свой user id
   hermes gateway start     # бот сразу оживает
   ```
   (То же можно сделать в приложении Hermes в разделе подключений/Connections.)

Теперь напиши своему боту в Telegram — он поздоровается и спросит, как его называть.

## Шаг 5. (По желанию) Умнее думать — Claude

Чтобы ассистентка думала на Claude вместо бесплатной модели: в чате с ботом набери
`/model` и выбери **Claude Sonnet 4.6** (баланс) или **Claude Opus 4.8** (максимум).
Это платно (через тот же Nous Portal — нужен платный тариф, или свой
`ANTHROPIC_API_KEY` через `hermes model`). Менять можно в любой момент.

## Шаг 6. (По желанию) Чтобы работала 24/7

Telegram отвечает, только пока машина включена. Чтобы всегда:
- держи компьютер включённым, **или**
- поставь Hermes на дешёвый VPS (Hetzner ~€4.5/мес, DigitalOcean) — те же шаги по
  SSH, и `hermes gateway` держи запущенным (как сервис/в фоне).

---

## Проверка

1. Бот в Telegram отвечает, тон тёплый и живой (не «как бот»), не больше 3 приоритетов.
2. Скажи «запомни: проверить память» → потом спроси «что я просила запомнить?» —
   должен вспомнить.
3. Без твоего «да» ничего не отправляет/не публикует/не удаляет.

## Если что-то не так

- Бот молчит → запущен ли `hermes gateway start`; верный ли твой user id; жив ли
  токен бота.
- Ошибка модели/доступа → проверь вход в Nous Portal (или баланс/тариф для Claude).
- «Звучит как бот», сахарит → открой `~/.hermes/SOUL.md`, секцию запретов, усиль
  под себя (см. `personalize/GUIDE.md`).
