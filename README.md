# Hermes — твой личный AI-ассистент, который живёт у тебя

Hermes — это каркас персонального ассистента (как Siri, только умнее и полностью
твой): он ведёт твой день, ловит мысли в инбокс, делает утренние и вечерние
сводки, помнит твои цели и не сливает личное наружу.

Это **не приложение, которое надо «скачать и запустить»**. Это набор инструкций
для ИИ. Ты отдаёшь эту папку своему Claude — и **он сам ставит ассистента тебе**:
задаёт пару вопросов, всё настраивает, проверяет, что работает. Тебе не нужно
уметь программировать.

> Этот репозиторий — **чистый шаблон**. Здесь нет ничьих личных данных. Твои
> ответы, цели и заметки появятся уже у тебя на компьютере, **вне** этой папки, и
> никуда не уедут.

---

## Что ты получишь

- **Общение в Telegram** — пишешь ему с телефона как обычному боту, а внутри —
  полноценный агент с твоей памятью и навыками (не болталка).
- **Утренний бриф** — план дня, фокус, верхние 3 задачи, настрой.
- **Инбокс на лету** — кидаешь мысль/идею/задачу, ассистент раскладывает по местам.
- **Вечерний разбор** — что получилось, инсайт, благодарность, план на завтра.
- **Память** — он помнит, кто ты, твои цели и повторяющиеся паттерны.
- **Границы** — рабочие дела и глубоко-личное не смешиваются; внешние действия
  (письма, публикации) — только с твоего «да».

## Как поставить (3 шага)

1. **Скачай эту папку.** На странице репозитория: зелёная кнопка **Code → Download
   ZIP**, распакуй. (Или `git clone`, если умеешь.)

2. **Открой папку в Claude Code.** Если Claude Code ещё не стоит — не страшно,
   ассистент по установке поможет и с этим (см. ниже «Совсем с нуля»).

3. **Вставь своему Claude одну фразу:**

   ```
   Прочитай INSTALL.md и поставь мне Hermes на сервер 24/7 на моей подписке
   Claude. Я не технарь — объясняй просто, делай за меня всё, что можешь, и
   спрашивай только то, что реально нужно от меня (регистрация, оплата, токен).
   ```

Дальше тебя ведёт сам агент. Он спросит, как тебе удобнее (на своём компьютере
или на сервере 24/7), всё развернёт и в конце покажет первый утренний бриф.

### Совсем с нуля (нет Claude Code)

Claude Code — это бесплатная программа от Anthropic, которая запускает ИИ-агента
у тебя в терминале. Установка — одна команда; нужен [Node.js](https://nodejs.org)
18+ и ключ/подписка Anthropic. Открой [INSTALL.md](INSTALL.md) — там первый
раздел ровно про это, и любой ИИ-чат (хоть Claude в браузере) проведёт тебя по
нему.

---

## Где это работает

Рекомендованный способ — **свой сервер 24/7**, чтобы Telegram отвечал с телефона
всегда, на **подписке Claude** (без оплаты за токены):

- **Сервер:** маленький VPS с 2+ ГБ памяти. По цене лучший — **Hetzner CX22**
  (4 ГБ, ~€4.5/мес); если с регистрацией сложно — **DigitalOcean** (~$12/мес).
- **Модель:** твоя подписка **Claude Pro** (этого хватает для личного ассистента,
  Max не нужен) — отдельно за токены платить не нужно. API-ключ тоже работает, но
  не обязателен.

Общаешься с ассистентом в **Telegram**, как с обычным ботом. Всё по шагам ставит
агент по [INSTALL.md](INSTALL.md). Хочешь сперва бесплатно пощупать на своём компе
без сервера — тоже можно (см. конец INSTALL).

## Сделать ассистента своим

Голос, оптика, твои цели и паттерны — всё настраивается. Это и есть самое
ценное: не безликий бот, а ассистент под тебя. Как собрать — в
[personalize/GUIDE.md](personalize/GUIDE.md). Агент по установке проведёт тебя и
по этому шагу.

---

## Приватность

- Личное (кто ты, цели, заметки) живёт в **живом доме агента у тебя на машине**,
  а не в этой папке. На GitHub оно не попадает.
- Никаких токенов, паролей и ключей в файлах — для них есть переменные окружения.
- Ассистент не отправляет письма, не публикует и ничего не удаляет без твоего
  подтверждения. Правила зашиты в `home/AGENTS.md`.

## Что внутри

```

├── README.md            ← ты здесь
├── INSTALL.md           ← инструкция, по которой агент тебя ставит

    ├── memories/        ← кто ты + долгая память (шаблоны, заполняешь сам(а))
    ├── workspace/       ← инбокс, задачи, идеи, рефлексии, папка брифов
    └── telegram/        ← мост: общение с ассистентом из Telegram (bot.py)
```
