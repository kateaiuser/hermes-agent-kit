# Индекс памяти

> Долгая память ассистента: по одной строке на факт, который стоит помнить между
> сессиями. Сам файл с фактом можно держать рядом; здесь — короткий указатель.
> Формат строки: `YYYY-MM-DD — факт (хук, по которому его узнать)`.
